Class선택자

Class선택자는 특정 클래스(Class)에 속하는 요소를 선택합니다.

아이디와 비슷한 기능을 하지만,

클래스는 여러 태그를 하나의 클래스로 묶을 수도 있고 태그 하나에 여러 클래스 이름을 붙여 줄 수도 있습니다.

        >> style 적용된 내용은 이하..
        
        .cs_fc {
            color: rebeccapurple;
        }
        .cs_fbg {
            background-color: yellowgreen;
        }
